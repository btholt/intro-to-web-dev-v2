## Welcome

Welcome to the code behind the [course website][website] for the [Introduction to Web Development v2][course] course on Frontend Masters.

## Development

To develop locally

1. git clone this repo
1. `npm install`
1. `npm run develop`

To publish

1. `npm run publish`
